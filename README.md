# Grammar and Spell Checker 

## 📌 Project Overview

Grammar and Spell Checker is a web application built using **Python Flask and Bootstrap** that helps users detect and correct spelling and grammar mistakes.

Users can either type text directly into the text box or upload a file containing text. The application processes the text and returns corrected spelling and grammar suggestions.

---

## 🚀 Features

* Check spelling mistakes in text
* Detect grammar errors
* Enter text manually for correction
* Upload text files for grammar checking
* Simple and responsive web interface
* Fast text processing

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/your-username/grammar-spell-checker.git
```

Go to the project folder:

```
cd grammar-spell-checker
```

Install required libraries:

```
pip install flask
```

---

## ▶️ Run the Application

Run the Flask application:

```
python app.py
```

Open your browser and go to:

```
http://127.0.0.1:5000
```

## 📖 How the Application Works

1. User enters text in the text box or uploads a file.
2. The text is sent to the Flask backend.
3. The backend processes the text using the SpellChecker module.
4. Spelling corrections and grammar mistakes are generated.
5. The corrected text and grammar suggestions are displayed on the webpage.
